## Project Structure

- `Frontend/` - React application built with Vite
- `Backend/` - Node.js API server

## Quick Start

### Backend
```bash
cd Backend
npm install
npm run dev
```

### Frontend
```bash
cd Frontend
npm install
npm run dev
```
